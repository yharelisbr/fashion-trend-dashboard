# Fashion Trend Dashboard

A fashion-tech data visualization project that analyzes trending fashion items, colors, and categories.  
Built using Python, Pandas, and Matplotlib.

- Data analysis  
- Trend forecasting  
- Visualization

- Building digital tools for retail & e-commerce

## Features

- Loads and analyzes real fashion trend data  
- Calculates average trend scores by category  
- Identifies top 8 fashion trends  
- Generates two visual charts:
  - Average Trend Score by Category
  - Top Fashion Trends
- Creates a complete HTML dashboard in the `/output` folder
